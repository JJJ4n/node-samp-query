# node-samp-query

Simplified Query API for SAMP

```
npm install samp-query
```

#### Usage

**Available options**

* host
* port - default: 7777
* timeout - default: 1000

```
var query = require('samp-query')

var options = {
	host: '94.23.166.205'
}

query(options, function (error, response) {
	if(error)
		console.log(error)
	else 
		console.log(response)
})
```
